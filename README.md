# 🚗 Automobile MPG Prediction using Polynomial Regression

## 📌 Project Overview
This project predicts the miles per gallon (MPG) of automobiles using Polynomial Regression.

The dataset contains automobile features such as:
- Cylinders
- Horsepower
- Weight
- Acceleration
- Displacement
- Model Year

The main objective of this project is to analyze automobile data, visualize relationships between variables, and build a machine learning model capable of predicting vehicle fuel efficiency.

---

# 📈 Model Performance

| Metric | Value |
|--------|--------|
| R² Score (Training) | 89% |
| R² Score (Testing) | 87% |
| Mean Squared Error (MSE) | 7.05 |
| Root Mean Squared Error (RMSE) | 2.66 |

---

# 📉 Result Analysis

- The model achieved strong prediction accuracy.
- Training and testing scores are very close, indicating minimal overfitting.
- Predicted MPG values closely match actual MPG values.
- Polynomial Regression successfully captured nonlinear relationships in the dataset.
